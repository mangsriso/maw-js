# MAW.JS FRONTEND (OFFICE) DEEP DIVE

## Architecture
- **React 19** + Vite + TailwindCSS 4.2
- **Zustand** (persistent store, localStorage + server sync)
- **Hash routing**: 9 views via App.tsx dispatcher
- **Real-time**: WebSocket with auto-reconnect

## Views (9 total)

| View | Route | Description |
|------|-------|-------------|
| Fleet | `#fleet` | Sortable agent list, grouped by session |
| Office | `#office` | Room grid with agent cards |
| Mission Control | `#mission` | Zoomable SVG map |
| Orbital | `#orbital` | 3D concentric rings (Three.js) |
| Overview | `#overview` | Terminal grid tiles |
| Terminal | `#terminal` | Full-screen xterm.js |
| Chat | `#chat` | Live/Timeline/Threads modes |
| Config | `#config` | Monaco editor for JSON configs |
| Worktrees | `#worktrees` | Git worktree hygiene |

## Agent Visual Design
- **Chibi avatars**: SVG, hash-based features (ears/antenna, eye style)
- **Status FX**:
  - Idle (#666): ground shadow only
  - Ready (#4caf50): subtle glow + ground radiance
  - Busy (#fdd835): rotating ring + sparkles + typing animation
- **Animations**: saiyan-outer, saiyan-aura, sparkle, chibi-spin, typing-arm
- **Colors**: 12-color deterministic palette from agent name hash

## State Management (Zustand)
- recentMap: last active agents (30-min TTL)
- sleptTargets: paused agents
- sortMode, grouped, collapsed: UI preferences
- asks: inbox items (50 max, auto-dismiss)
- Persistence: localStorage + debounced server sync (1s)

## Real-Time Pipeline
```
WebSocket → useSessions hook
  → sessions (5s), capture (50ms), previews (2s)
  → feed events → status decay (busy→ready 15s, ready→idle 60s)
  → ask detection (Stop + Notification pair)
```

## Interactive Features
- **Terminal Modal**: xterm.js PTY, Alt+1-9 agent switching
- **Jump Overlay**: Fuzzy search (Cmd+J / Ctrl+K)
- **Inbox**: Ask cards with approve/reject/reply
- **Chat**: 3 modes (Live streaming, Timeline grouped, Threads paired)

## New Features (Recent)
- Chat system (chat/ subdirectory): ChibiAvatar, ChatBubble, ThreadCard, useChatLog
- Worktree View: status badges, cleanup integration
- Config View: Monaco editor, file toggle, dirty tracking
- Inbox/Asks: type system (input/attention/plan), auto-dismiss

## Performance
- React.memo on all views + cards
- IntersectionObserver for visible-only preview subscriptions
- Lazy loading: XTerminal (dynamic import)
- Feed: circular buffer (200 events max), Map-based state
- ResizeObserver for dynamic canvas sizing

## Design Language
- Dark glassmorphic (#020208 backgrounds)
- Neon accents per agent (deterministic)
- Micro-interactions: 300-500ms transitions, active:scale-95
- Custom scrollbar hiding, text selection styling
